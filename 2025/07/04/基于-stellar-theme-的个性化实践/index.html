使用 NPM 安装 Stellar 后如何修改字体样式？

在使用 Hexo 构建博客并安装 Stellar 主题的过程中，如果你使用了如下命令：

你可能会发现一个问题：

在 themes/ 文件夹下看不到 Stellar 主题的文件，无法直接修改模板或 CSS ，这是因为通过 NPM 安装主题的方式会将主题文件存放在项目的 node_modules/ 文件夹中，而不是 themes/ 下。

以 Stellar 为例，实际路径是：node_modules/hexo-theme-stellar/

这种结构的好处是可以轻松升级主题，但缺点是无法直接修改主题文件（否则更新时改动就会被覆盖），因此，我们需要采用 Hexo 官方推荐的方式，用 _config.stellar.yml 来“注入”我们自己的样式，而不是直接改源码。

步骤一：添加自定义 CSS 文件

在你的博客根目录下，进入 source/css/ 文件夹，新建一个名为 custom.css 的文件。

📁 路径结构为：your-blog/source/css/custom.css
